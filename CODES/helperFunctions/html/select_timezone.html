function [timezone] = select_timezone

% select_timezone return timezone from user selection (based on MATLAB timezones)

Syntax

[timezone] = select_timezone

Description

'timezone' is a string variable of a user-selected Internet Assinged Numbers Authority (IANA) time zone accepted by the datetime function.

Can be used to define the timezone of a MATLAB datetime variable: datetime_variabel.TimeZone = timezone.

See href="matlab:web('https://www.mathworks.com/help/matlab/ref/timezones.html')" MATLAB timezones</a> for list of timezones.
